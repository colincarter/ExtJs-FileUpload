ExtJs-FileUpload by colincarter

A file uploader for ExtJS

A File uploader for ExtJS. Uses the php pecl extension uploadprogress.

Dependencies

ExtJS Library from extjs.com

Install

Copy extjs library to a folder called ext. Install uploadprogress pecl extension (sudo apt-get install php5-dev && sudo pecl install uploadprogress), Add extension=uploadprogress.so to the end of the apache php.ini (/etc/php5/apache2/php.ini)
